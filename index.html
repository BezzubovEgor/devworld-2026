🎯 Kinie-Winny's Day 2 Recommendations

🔴 09:30 — Giving AI Agents a Wallet (Keynote)
Kevin Lewis — agent payments продолжение
Прямое продолжение вчерашнего "Teaching Agents to Pay" и AP2/x402. Keynote = главный доклад слота. Тот же Kevin Lewis потом ведёт workshop "How to Feed AI Agents with Data" — можно подойти.
🔴 11:00 — Auth for AI Agents (Auth0)
Jeremy Durand — AuthN/AuthZ для агентов
Auth0 для AI-агентов — критическая тема. Вчера был MCP security, сегодня — как агенты аутентифицируются. Если Kineto создаёт агентов, им нужен identity layer.
🔴 11:00 — Email as Agent Runtime
Confidence Okoghenun — inbox = agent runtime
Провокационная идея: email — лучший протокол для AI-агентов. Async, structured, universal. Конфликтует с Auth0 talk — выбирай один. Я бы выбрал Auth0 (практичнее), но этот оригинальнее.
🔴 11:30 — Fireside: CEO of GitBook
Future of Documentation with AI
CEO-level fireside (как Netlify вчера). Документация + AI = agent-facing docs. GitBook = major docs platform. Networking opportunity #1 Day 2.
🔴 12:00 — Structure Context for Smarter Agents
Nyah Macklin — context engineering
Структурирование контекста = разница между тупым и умным агентом. Прямо применимо к Kineto — ты строишь платформу где люди создают агентов.
🔴 13:30 — AI for Everything After Code
Martin Reynolds — ship fast, stay safe
Main Stage. AI не только пишет код — деплой, мониторинг, security, incident response. Практический взгляд на полный цикл.
🔴 14:00 — GitLab: AI Adoption in Engineering Teams
Viktor — enterprise-scale кейс
GitLab показывает как реально внедряют AI-агентов в engineering teams. Enterprise case study = конкретные цифры и проблемы. Keynote.
🟠 10:00 — Agentic AI + Event-Driven Architecture
Enterprise challenges
AI agents + EDA — как интегрировать агентов в event-driven системы. Архитектурный доклад.
🟠 14:30 — AI-Native: Next Revolution after Cloud-Native
Pini Reznik — стратегический взгляд
Cloud-Native → AI-Native. Стратегическое видение куда движется индустрия. Kineto = AI-native by design.
🟠 15:00 — Rust-Inspired TypeScript
Rijk van Zanten — Result types, pattern matching
Rust паттерны в TS — Result, Option, exhaustive matching. Прямо применимо к твоему коду. Приятный технический перерыв после AI-марафона.
💡 Красная нить Day 2: Agent Identity (wallet + auth) → Agent Context (structure + docs) → Agent Adoption (GitLab + SDLC + change management). Более практический день чем Day 1 — меньше vision, больше implementation.
🤝 Networking: Два networking mixer'а в конце дня — 15:30 (Diverse Leaders) и 16:30 (Leaders & Founders). 16:30 = must-go. Founders mixer на закрытие конференции = лучшие контакты.
